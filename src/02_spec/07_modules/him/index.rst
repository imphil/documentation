Host Interface Module (HIM)
===========================

.. figure:: ../../img/debug_module_him.*
   :alt: Host Interface Module
   :name: fig:debug_module_him

   Host Interface Module

The *Host Interface Module (HIM)* interfaces the Debug Interconnect with the host.
It converts Debug Packets into a representation suitable for off-chip transfer, and injects packets arriving from the host into the Debug Interconnect.
